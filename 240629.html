Windows 8.1

发布于2013年10月17日

系统介绍

Windows 8.1是微软公司推出的最新一代操作系统，相较于之前的Windows 8，Windows 8.1在界面、功能和安全性等方面都有了很大的改进和提升。
Windows 8.1采用了全新的Metro界面，这种界面风格更加简洁明了，用户可以更轻松地浏览和使用各种应用程序。
Windows 8.1也进行了一些性能上的优化，例如提高了系统的响应速度和稳定性，降低了功耗等。这使得Windows 8.1成为一款非常优秀且稳定的操作系统。
总之，Windows 8.1是一款具有划时代意义的新型操作系统，它不仅带来了全新的界面和功能，同时也为用户提供了更好的使用体验和安全保障。

系统改动内容

运行库集成：.net3.5，DX9，VC2005-2022，.net4.8
精简优化内容：操作中心、Windows Anytime Upgrade(用于 Windows 低级版本升级到高级版版）、.NET assembly 缓存、轻松访问主题、文件历史、文件管理器 (OneDrive & Photos)、游戏资源管理器、提示（Win10 1607之前叫“入门”）、Intel Indeo 编码器、Windows 光盘映像刻录工具、Manifest 备份 (WinSxS\Backup)、手动安装、Media Center（媒体中心）、BingFinance、BingFoodAndDrink、BingHealthAndFitness、BingMaps、BingNews、BingSports、BingTravel、BingWeather、HelpAndTips、Media.PlayReadyClient.2、Office.OneNote、Reader、SkypeApp、WindowsAlarms、Windows Mail、WindowsReadingList、WindowsScan、WindowsSoundRecorder、XboxLIVEGames、ZuneMusic、ZuneVideo、轻松传送、其他主题、Windows Reader (PDF)、屏保、安全中心、声音方案、手动功能库(sources/sxs)、语音识别、语音 TTS、Superfetch（超级预读取）、系统恢复、缓存与临时文件、视频压缩管理器(VCM)编码器、墙纸 (主题)、Windows Defender、Windows TIFF IFilter(光学字符识别 OCR )、Windows Recovery（Windows 恢复）、Windows 系统评估工具 (WinSAT)、还有许多杂七杂八的内容（可以看视频制作过程，大多都用Dism++优化）
